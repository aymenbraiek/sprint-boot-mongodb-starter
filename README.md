# Spring Boot Demo
An attempt to create a basic spring boot demo with MongoDB backend

## Requirements

- Java 8+
- Gradle 4+
- MongoDB localhost server

## Run

```sh
$ ./gradlew bootRun
```

## Build

```sh
$ ./gradlew build
```

## Run

```
$ java -jar build/libs/gs-spring-boot-0.1.0.jar
```
